Severity Explanation - Suricata Alerts
=======================================

SEVERITY LEVELS:
----------------

Severity is ONLY shown for ALERT events (actual security threats detected).

For normal traffic (flow, netflow, stats), severity shows "-" (not applicable).

ALERT SEVERITY LEVELS:
---------------------

1 = LOW SEVERITY (Green badge)
   - Minor security issues
   - Informational alerts
   - Suspicious but not dangerous

2 = MEDIUM SEVERITY (Orange badge)
   - Moderate security concerns
   - Potentially malicious activity
   - Should be reviewed

3 = HIGH SEVERITY (Red badge)
   - Critical security threats
   - Active attacks detected
   - Immediate attention required

4 = CRITICAL SEVERITY (Red badge)
   - Severe security breaches
   - Active exploitation attempts
   - System compromise attempts
